Board games

A board games dataset. A dataset contains info about multiple board games. Every game attribute is described in a dataset description. Attributes which are type of object or array of objects are described in their own tables. Each attribute represents a key in json dataset file.

complexity Average rating of a game complexity on a scale from 1 to 5 numeric
